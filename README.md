Signal Processing and Data Compression Repository
Description:
This repository contains a collection of laboratory work, exercises, and scripts written in Python and MATLAB related to signal processing, data compression, and communication systems.
The content is intended for academic and educational use, mainly for students studying telecommunications, signal processing, or related fields.

*Main Topics Covered:

+Data compression algorithms

+Source coding techniques

+Error detection and correction

+Fourier transform and spectral analysis

+Audio signal processing

+Speech and voice analysis

+Digital watermarking
